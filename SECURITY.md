# Security Policy

## Supported Versions

| Version | Supported |
|----------|-----------|
| 2.x | Yes |
| 1.x | No |

## Reporting a Vulnerability

If you discover a security vulnerability, please do not disclose it publicly.

Instead, open a private GitHub Security Advisory or contact the maintainer.

Please include:

- Description
- Steps to reproduce
- Expected behavior
- Screenshots if applicable

I will investigate the report as soon as possible.
